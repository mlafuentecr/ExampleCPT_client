# Example CPT WordPress Plugin

## Description

The Example CPT WordPress Plugin is a custom plugin that adds a new custom post type (CPT) called "Example CPT" to your WordPress website. This plugin allows you to easily create and manage custom posts with a specific set of attributes tailored to your needs.

## Features

- Adds a new custom post type called "Example CPT" to your WordPress site.
- Provides an intuitive user interface to create and edit custom posts with custom fields.
- Allows you to define and customize the attributes of the "Example CPT" posts.
- Seamless integration with the WordPress dashboard and user interface.

## Installation

To install the Example CPT WordPress Plugin, follow these steps:

1. Download the plugin ZIP file from [mariolafuente.com/example-cpt-plugin](https://eddy.sitecr.com/ExampleCPT_client.zip).
2. Log in to your WordPress admin panel.
3. Navigate to "Plugins" > "Add New."
4. Click on the "Upload Plugin" button at the top of the page.
5. Choose the ZIP file you downloaded in step 1 and click "Install Now."
6. Once the installation is complete, click on "Activate Plugin" to activate the Example CPT WordPress Plugin.

## Usage

1. After activating the plugin, you'll find a new menu item called "Example CPT" in the WordPress admin sidebar.
2. Click on "Example CPT" to access the "Example CPT" custom post type section.
3. To create a new "Example CPT" post, click on "Add New."
4. Fill in the necessary details for your custom post, including the custom fields defined for this post type.
5. Click on "Publish" to save your "Example CPT" post.

## Customization

The Example CPT WordPress Plugin allows you to customize the attributes of the "Example CPT" custom post type according to your requirements. To do so, follow these steps:

1. In the WordPress admin sidebar, go to "Settings" > "Example CPT."
2. Here, you can define and manage the custom fields that will be available for the "Example CPT" posts.
3. Add, remove, or modify the custom fields as needed.
